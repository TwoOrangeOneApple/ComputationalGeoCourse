    "# Initial setup\n",
    "\n",
    "\n",
    "1. Install miniconda for you architecture:  https://docs.conda.io/en/latest/miniconda.html\n",

    "## For MacOS\n",
    "\n",
    "a. Start a terminal and type:\n",
    "\n",
    "```\n",
    "conda install git\n",
    "```\n",
    "\n",
    "b. cd to your home directory and make a folder called repos\n",
    "\n",
    "```\n",
    "mkdir repos\n",
    "cd repos\n",
    "```\n",
    "\n",
    "c. clone the course repository and cd into it\n",
    "\n",
    "```\n",
    "git clone https://github.com/eldadHaber/ComputationalGeoCourse\n",
    "cd ComputationalGeoCourse\n",
    "```\n",
    "\n",
    "d. checkout the pha branch\n",
    "\n",
    "```\n",
    "git checkout -b pha origin/pha\n",
    "```\n",
    "\n",
    "e. cd to the conda folder and create and activate the e213 environment\n",
    "\n",
    "```\n",
    "cd conda\n",
    "conda env create -f environment.yml\n",
    "conda activate e213\n",
    "```\n",
    "\n",
    "f. change back to the notebooks folder and start jupyter\n",
    "\n",
    "```\n",
    "cd ../notebooks\n",

    "```\n",
