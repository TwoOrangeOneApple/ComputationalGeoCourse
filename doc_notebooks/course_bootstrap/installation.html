Initial setup¶

  1. Install miniconda for you architecture: https://docs.conda.io/en/latest/miniconda.html

For MacOS¶

  1. Start a terminal and type:

conda install git

  1. cd to your home directory and make a folder called repos

mkdir repos
cd repos

  1. clone the course repository and cd into it

git clone https://github.com/eldadHaber/ComputationalGeoCourse
cd ComputationalGeoCourse

  1. checkout the pha branch

git checkout -b pha origin/pha

  1. cd to the conda folder and create and activate the e213 environment

cd conda
conda env create -f environment.yml
conda activate e213

  1. change back to the notebooks folder and start jupyter

cd ../notebooks
jupyter notebook
